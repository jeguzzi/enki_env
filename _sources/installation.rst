============
Installation
============

.. code::

   pip install git+https://github.com/jeguzzi/enki_env

This will install dependencies `pyenki <https://jeguzzi.github.io/enki/>`_, `gymansium <https://gymnasium.farama.org>`_, and `pettingzoo <https://pettingzoo.farama.org>`_.

Add option ``[torchrl]`` to additionally install `torchrl <https://docs.pytorch.org/rl>`_ and `benchmarl <https://github.com/facebookresearch/BenchMARL>`_.
